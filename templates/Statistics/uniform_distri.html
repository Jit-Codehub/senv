Steps to calcualate uniform distribution probability

probability density Formula-

if a ≤ X ≤ b then f(x,a,b) = 1 / (b - a)


if a < X and b < X then f(x,a,b) = 1 / (b - a) then f(x,a,b) = 0


lower cumulative distribution Formula-

(x - a) / (b - a)


upper cumulative distribution Formula-

(b - x) / (b - a)


where:
  • percentile x - probability of X lying in an interval (a, b)
  • uniform interval a, b - Uniform distribution range
