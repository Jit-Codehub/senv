Mean Deviation - Definition

Mean Deviation is nothing but the statistical measure used for determining the average deviation from the mean value of the given data set.

Mean Deviation Formula

The formula for calculating the Mean Deviation of a given set of data is given by the expression ∑i = 0n |xi − μ| / N

  • Where, Σ represents the addition of values
  • X represents each value in the data set
  • μ is the mean of data set
  • N is the Number of Values
  • | | represents the absolute value, which ignores the “-” symbol

How to Find Mean Deviation?

Follow the below listed procedure to quickly find the mean deviation of the given set of numbers manually. They are along the lines

  • Firstly, identify the given set of numbers.
  • Then, find out the mean of the given data set i.e. sum of numbers/total numbers in the set.
  • Compute the difference between each terms in the data set and mean to obtain the new values.
  • At last, find the mean of new numbers to obtain the mean deviation value.

Question 1: Find the Mean Deviation of 4, 8, 8, 10, 12, 11, 15, 16?

Solution:

Given numbers are 4, 8, 8, 10, 12, 11, 15, 16

Mean = Sum of items/ Number of items

= 4 + 8 + 8 + 10 + 12 + 11 + 15 + 16 / 8 = 84 / 8

μ = 10.5

Mean Deviation = ∑i = 0n |xi − μ| / N

= |(10.5 - 4) + (10.5 - 8) + (10.5 - 8) + (10.5 - 10) + (10.5 - 12) + (10.5 - 11) + (10.5 - 15) + (10.5 - 16)| / 8

= (6.5) + (2.5) + (2.5) + (0.5) + (1.5) + (0.5) + (4.5) + (5.5) / 8

= 24 / 8

= 3

So, Mean is 10.5 and Mean Deviation is 3
