What is meant by Minimum Number?

Minimum Value is defined as the smallest or least value in the given set of numbers. In short it is represented by min or minima. We can arrange the set of numbers in the form of x1 ≤ x2 ≤ x3 ≤ ... ≤ xn. The first one is the smallest value x1.

How to Find the Minimum Number Manually?

To find the minimum value follow the simple steps listed below. They are along the lines

  • First and foremost step to keep in mind is to sort the numbers.
  • Arrange the numbers in ascending order from least to greatest.
  • After arranging in ascending order the number that is smaller is the minimum number of the given data set.

Example:

Find the minimum of the number from the given data set: 45,32,323,453,12,621?

Solution:

Given numbers are 45,32,323,453,12,621

Firstly, sort the given numbers in ascending order.

Thus the list becomes like this: 12, 32, 45, 323, 453, 621

From the order of numbers, 12 is the minimum number.

Therefore, the minimum of the given data set 12, 32, 45, 323, 453, 621 is 12.
