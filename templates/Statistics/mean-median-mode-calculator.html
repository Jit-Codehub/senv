What is Meant by Mean Median Mode?

Mean Median Mode is three measures of central tendency. The definitions of these measures are provided here:

Mean:

Mean is the arithmetic average of a data set. Mean is also known as Average. The definition of the mean is the sum of given data by a total number of data. For example, lets take the data set as 2, 5, 7, 8, 9, 10, 11. The mean of this data set is 7.4.

Median:

The definition of median is the middle number in a data set when the numbers are listed in either ascending or descending order. When the data set is of two numbers then the middle number or median is the average of those 2 numbers. For instance, consider the above example and the median is 8 (a middle number of a data set).

Mode:

A number in a data set that appear most often is called mode. However, the difference between the highest and lowest values in a data set is called range. For example, let's consider the data set 1,4,5,3,2,1,4,6. The mode of the data set is 1 and also 5.

Detailed Steps on How to Find Mean Median Mode?

The step-by-step process to find the mean, median, mode values are explained here accordingly. Take a look at the simple & manual process and find the solutions easily & quickly.

Mean Calculation Process:

  • Firstly, find the sum of values in a data set.
  • Count the values in the data set and find the total
  • Now, divide the sum of values by the total values count to calculate the mean of a data set.

Median Calculation Process:

  • Initially, the given data set should be arranged in ascending order from the lowest value to the highest value.
  • Look for the middle number in the arranged data as it is called the median.
  • In case, the data set is having two middle places then the median is calculated by finding the average of those two values.

Mode Calculation Process:

  • At first, arrange the number in asc order.
  • Now, the values that are repeated in the list will be arranged side by side.
  • Count how many times each value appears.
  • If any of the values appear most often, then that number is the mode of the data set.

Mean = Sum of the Given Data / Total Number of Values

Range:

It is the difference between the maximum and minimum values.

Range = xmax - xmin

Minimum:

Which is the lowest value in the data set. Arrange the data set in ascending order.

xmin = min(xi)i=1n

Maximum:

It is the highest value available in the data set. Arrange the data set from the lowest to highest values.

xmax = max(xi)i=1n

Quartiles:

Q1 is the lower half of the median, Q2 is the median, Q3 is the higher half of the median.

Interquartile Range IQR:

Range from Q1 to Q3 is the IQR.

IQR = Q3 - Q1
