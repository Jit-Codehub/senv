What is meant by Maximum Number or Maxima?

The Largest Number among a given set of data is known as Maximum Number. We use the symbol Max to denote the Maximum Number and the plural form is Maxima. For Example if a data includes set of numbers 4, 16, 78, 91 then the maximum number among the list is 91.

How do you find the Maximum Value of a Number Set?

Go through the steps mentioned below to determine the maximum number among a given data set of numbers manually. They are as below

  1. Initial step is to sort the given data set from low to high i.e. in ascending order.
  2. As you have sorted it is easier to find the largest value from the set of numbers.
  3. The last value in the data set is nothing but the maximum number of the given data set.
