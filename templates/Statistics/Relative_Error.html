What is the Relative Error?

The relative error is defined as the ratio of the absolute error of the measurement to the actual measure. This method is used to determine the magnitude of the absolute error in terms of the actual quantity. It is alo called as relative uncertainty or approximation error.

Absolute error is the difference between the true quantity and measured quantity. The formula to calculate absolute error and relative error are along the lines:

Absolute Error = Actual Value - Measured Value

Relative Error = (Actual Value - Measured Value)/Actual Value = Absolute Error/Actual Value

Steps to Calculate Relative Error

Go through the detailed step by step process to find the relative error of the quantity quickly.

  • Get the actual value, measured value from the question.
  • Subtract measured value from the actual value.
  • Divide the difference by the actual value to get the relative error.

Solved Questions on Relative Error

Question 1.Find the absolute and relative errors of the approximation 168.5 to the value 120.1

Solution:

Absolute Error = Actual Value - Measured Value

Absolute error = 168.5 - 120.1 = 48.4

Relative Error = (Actual Value - Measured Value)/Actual Value

Relative error = 168.5 - 120.1/120.1

= 48.4/120.1 = 0.402

Therefore, relative error is 0.402.

FAQ's on Relative Error Calculator

1. What is a relative error example?

An example of a relative error is an error of 1 cm would be a lot if the total length is 20 cm, but insignificant if the length was 10 km.

2. How do you calculate relative random error?

The simple step to obtain relative error or fractional error is to divide absolute error in the quantity by the quantity itself.

3. Is high relative error Good?

The relative error is useful when you want to compare things that are measured in different units. The high relative error is not good.

4. What is a relative error and absolute error?

Absolute error is the difference between measured and the actual value of a quantity. The absolute error is inadequate due to the fact that it does not give regarding the importance of the error. The relative error is defined as the ratio of the absolute error of the measurement to the actual measurement.
