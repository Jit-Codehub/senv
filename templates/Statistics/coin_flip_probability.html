Coin Toss Probability Formula

Probability of an event is nothing but the ratio of number of possible outcomes to the total number of outcomes. The same logic applies to coin flip probability as well.

Probability = Number of Possible Outcomes/Total Number of Outcomes

Whenever you toss a coin there is a chance of only two possible outcomes. Using the basic probability formula we can get the coin toss probability formulas as follows

  • Probability of getting a head P(Head) = P(H) = 1/2
  • Probability of getting a head P(Tail) = P(T) = 1/2

Since the outcomes of a coin toss are independent, total number of outcomes is equal to the total number of trails times the number of possible outcomes in a single trail i.e. 2. The number of favorable outcomes is nothing but the number of favorable combinations. We can obtain the coin toss probability using the formula

Coin Toss Probability = nCh/2n where n, is the total number of trails and h is the number of heads in n number of trails.

Coin Flip Probability Example

Question 1: What will be the probability of getting 3 heads in 5 coin tosses?

Solution:

Number of trials (n) = 5

Total possible outcomes = 2n = 25 = 32

Number of heads (h) = 3

Total number of favourable outcomes = 5C3 = 10

Now probability = 10/32 = 0.3125
