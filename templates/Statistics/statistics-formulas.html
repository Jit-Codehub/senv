  • Mean x̄ = Σxi / N
    • 1. Arithmetic Mean For Sample Data Set:
      Xmean = (∑i = 1nxi) / n = [x1 + x2 + x3 + . . . + xn] / n
      AM For Population Data Set:
      μ = (∑i = 1nxi) / n = Sum / size = [x1 + x2 + x3 + . . . + xn] / n
    • 2. Geometric Mean: It is defined as the product of all data values raised to the power of the reciprocal of the count of data values. Just, the nth root of the product of data values is the geometric mean.
      GM = (Πi = 1n)^1/n = (n)√(x1.x2.x3. . . . xn)
    • 3. Harmonic Mean: Harmonic mean is n times the reciprocal of the sum of the reciprocals of the data values. Or, Divide n by the sum of reciprocals of all data values.
      HM = (n) / [∑i = 1n(1/xi)] = (n) / [1/x1 + 1/x2 + 1/x3 + . . . + 1/xn
  • Median = (N+1)th / 2 term; when N is odd
    [ Nth / 2 term + ( N / 2 + 1)th term ] / 2 ; when N is even
  • Mode = The value in the data set that occurs most frequently
  • Variance = Σ(xi – x̄)^2 / N
  • Standard Deviation = √Variance
  • Relative Standard Deviation for Population Data (RSD) = [(100 * σ) / μ]%
  • Relative Standard Deviation for Sample Data (RSD) = [(100 * s) / μ]%
  • Z-score = (x - μ) / σ or z = (x mean - μ) / σ/√n
  • Mean Deviation (Mean Absolute Deviation MAD) = (∑i = 1n |xi - μ|) / n
  • Root Mean Square (RMS) = √[(∑i = 1n (xi)²) / n ]
  • Standard Error of the Mean = SEμ = (σ / (√n))
  • Sum of Squares (SS) = ∑i = 1n (xi - μ)²
  • Skewness γ1 = (∑i = 1n (xi - μ)3) / (n * σ3)
  • Kurtosis = β2 = (∑i = 1n (xi - μ)4) / (n * σ4)
  • Coefficient of Variation ( CV ) = σ / μ
  • Kurtosis Excess = α4 = [((∑i = 1n (xi - μ)4) / (n * σ4)) - 3]
  • Outliers formulas are Upper Fence = Q3 + 1.5 x IQR and Lower Fence = Q1 - 1.5 x IQR
  • Interquartile Range IQR = Q3 - Q1
  • Weighted Mean | Weighted Average: Xweighted_mean = ∑i = 1n(wi * Xmean) / ∑i = 1nwi
  • Maximum = xn = max(xi)i=1n
  • Minimum = x1 = min(xi)i=1n
  • Range: R = Maximum - Minimum = xn - x1
  • Midrange: MR = (Minimum + Maximum) / 2 = (x1 + xn) / 2
  • Frequency: The number of occurrences for each data value in the data set is called frequency.
  • Count: It is the total number of values in the data set. This is also known as the size of the data set. Count = n = count(xi)i = 1n
  • Sum: The total of all data values in the data set is sum. Hence, sum = x1 + x2 + x3 + . . . + xn = ∑i = 1nxi
  • Percentile: Arrange all data values in ascending order i.e x1 ≤ x2 ≤ x3 ≤ . . . ≤ xn
    • Find the rank for the percentile p using r = (p/100) * (n - 1) + 1
      If r is an integer then, data value at the r location is percentile p
      p = xr
    • If r is not an integer, p is interpolated using ri, the integer part of r and rf, the fractional part of r
      p = xri + rf * (xri+1 - xri)

Example 1:

In a kitchen, there are 30 participants. 4 were picked randomly and their total scores in the cooking are recorded as 34, 56, 64, 76. Find the standard deviation of their scores.

Solution:

Given N=4

Sample mean = 34+56+64+76 / 4 = 230/4 = 57.5

Variance = ∑(x−μ)^2/n

= (34-57.5)^2 + (56-57.5)^2 + (64-57.5)^2 + (76-57.5)^2 / 4

= 939/4

= 234.75

Standard deviation = √234.75 = 15.321

Example 2:

Find the mode of the set S = 1,2,3,2,3,2,3,4,2,4,5,2,6,7.

Solution:

Given S = 1,2,3,2,3,2,3,4,2,4,5,2,6,7

Arrange the data in ascending order:

1,2,2,2,2,2,3,3,3,4,4,5,6,7

Here 2 occurs a maximum of 5 times, therefore 2 is the mode of the set.
