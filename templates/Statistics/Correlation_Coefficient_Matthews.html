What is Matthews Correlation Coefficient? | Formula of MCC

A correlation coefficient is the measure of the strength of correlation, i.e the statistical connection between two variables. MCC describes how changing one variable value to affect another variable value. The correlation value lies between -1 to +1.

  • 0 indicates no relationship between a set of variables
  • -1 indicates an ideal relation i.e variables change in opposite direction
  • +1 indicates a perfect positive relation i.e variables change in the same direction

The formula to find Matthews correlation coefficient is given here.

Where,

  • TP (True Positive) is the outcome where the model correctly predicts the positive class
  • TN (True Negative) is the outcome where the model correctly predicts the negative class
  • FP (False Positive) is the outcome where the model incorrectly predicts the positive class
  • FN (False Negative) is the outcome where the model incorrectly predicts the negative class

Correlation Coefficient(Matthews) Example Questions

Find the Matthews Correlation Coefficient. When true positive is 170, true negative is 220, false positive is 450 and false negative is 330.

Solution:

Given that,

True positive TP = 170

True negative TN = 220

False positive FP = 450

False negative FN = 330

Matthews correlation coefficient formula is MCC = ((TP*TN) - (FP*FN))/√((TP+FP)*(TP+FN)*(TN+FP)*(TN+FN))

Substitute the values in the above formula.

MCC = ((170.0 * 220.0) - (450.0 * 330.0)) / √((170.0 + 450.0)*(170.0 + 330.0)*(220.0 + 450.0)*(220.0 + 330.0))

= -0.3287111728354338

Therefore, Matthews Correlation Coefficient is -0.3287111728354338.
