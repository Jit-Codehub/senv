What do you Mean by SMp(x) Distribution?

Probability distribution is a mathematical function that chooses you the probabilities of all possible outcomes.

In this function we have two properties:

  1. The value of a probability function should not be lower than zero.
  2. And, if you combine the function from minimum to maximum, the value of the probability density function will be equal to 1. It means that one of the outcomes must happen - there's zero chance that none of them should takes place.

How SMp(x) Distribution Works?

SMp(x) is a function which is very flexible and describes various probability distributions, irrespective of weather they are continuous or discrete. Here, we will see about the six parameters.

  • P(Xmin) is the lower limit value of x.
  • Xmax is the upper limit of value x.
  • ML is the value of x, when the probability distribution function reaches its maximum value.
  • Next, p₁ and p₂ are the powers of the model. Both of them should be greater than zero.
  • Max is the maximum of the SMp(x) function.

Conditions for SMp(x) Distribution

Once if we know the values of all the six parameters, you can calculate the problem based on below conditions or formulas.

If x < P(Xmin), then SMp(x) = 0

If, P(XMin) ≤ x ≤ ML then SMp(x) = [(x - P(Xmin))/(ML - P(Xmin))]^ p₁ * Max

If ML ≤ x ≤ XMax, then SMp(x) = [(Xmax - x) / (Xmax - ML)]^ p₂ * Max

If x > Xmax, SMp(x) = 0.
