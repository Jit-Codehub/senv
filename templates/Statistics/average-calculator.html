How to Find Average or Mean of Data Set?

Average is a number that represents all numbers in the set. Mean and average give the same values. Here are some simple steps to calculate the average of numbers easily.

  • Primarily, note down all the values in the given data set.
  • Count the values you have entered, as we can sum only upto thirty numbers.
  • Now, add all the values in the given data set.
  • Finally, divide the sum by the count of values in the set to get the average.

Average Formula

Average of a data set is described as the sum of all the values will be divided by the size of data set.

Mean (or) Average = xmean = (∑ni=1 xi) / n

Where,

i=1,2,3,4.....,n

n is the count of values in the set.

Average Calculation Examples

Question 1: Compute the average of the given data set {24, 43, 66, 44, 97, 4, 35}?

Solution:

Given, data set {24, 43, 66, 44, 97, 4, 35}

Count of Values = 7

Formula of Average (or) Mean = xmean = (∑ni=1xi) / n

= (24 + 43 + 66 + 44 + 97 + 4 + 35) / 7

= 313/7

= 44.71

∴ Average = 44.71, sum = 313 and count = 7
