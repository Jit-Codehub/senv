In statistics, the interquartile range is defined as the difference between the third quartile and first quartile of the data set. The semi-interquartile range is the measure of dispersion. Quartiles divide the data into 4 equal proportions. So, it forms 3 quartiles. They are as follows.

  1. First quartile is represented as Q1 and is also known as the lower quartile.
  2. Second quartile is represented as Q2.
  3. Third quartile is represented as Q3 and is also known as the upper quartile.

The formulas to find the interquartile range, quartiles are given below:

Interquartile range (IQR) = Upper quartile - lower quartile = Q3 - Q1

Where,

Third quartile Q3 = ¾(N + 1)

First quartile Q1 = ¼(N + 1)

N is the number of elements in the data set.

Steps to Calculate IQR (Interquartile Range)

Check out the step by step process on how to calculate the intermediate range of the data set easily.

  • Arrange the given data set in the increasing order and count the number of elements in it.
  • Find the median of the given numbers which is second quartile Q2.
  • Median equally cuts the given values into two equal parts. They are described as Q1 and Q3 parts.
  • The median of the numbers below the median is Q1.
  • The median of the numbers above the median is Q3.
  • Subtract Q1 from Q3 to get the interquartile range value.
  • Otherwise, put the values in the given Q1, Q3 formulas to obtain the answer.

Example

Question: Find the inter, quartile range of numbers {2,14,8,7,9,16,13,10,5}?

Answer:

Given data is {2,14,8,7,9,16,13,10,5}

The number of elements are N = 9

The formula to find first quartile Q1 = ¼(N + 1)

Q1 = ¼(9 + 1)

= 2.5

The formula to calculate third quartile Q3 = ¾(N + 1)

Q3 = ¾(9 + 1)

= 7.5

Interquartile range formula is IQR = Q3 - Q1

IQR = 7.5 - 2.5

= 5

Therefore, interquartile range of {2,14,8,7,9,16,13,10,5} is 5.

FAQ's on Interquartile Range Calculator

1. How do I calculate the IQR?

Follow these instructions to calculate the interquartile range of the data set. Get the data set and find the number of elements. Substitute the number of elements in the first and third quartiles formula. The difference between the third quartile and the first quartile is IQR.

2. What is Interquartile Range in statistics?

The interquartile range is defined as the difference between the third quartile and the first quartile. Quartiles are divided into 4 equal parts. So, there are 3 quartiles. The first quartile is Q1 or lower quartile, the second quartile is Q2 and the third quartile is Q3 or upper quartile.

3. How to find Q1 and Q3 from the data?

The formulas to find the first quartile and third quartile of the data set are here:

First quartile Q1 = ¼(N+1)

Third quartile Q3 = ¾(N+1)

4. Find the intermediate quartile for the data set {1, 8, 5, 6, 4, 2}?

Arrange the data in the ascending order i.e {1, 2, 4, 5, 6, 8}

The set of numbers n = 6

First quartile Q1 = ¼ (n+1)

= 1/4(6 + 1)

= 7/4 = 1.75

Third quartile Q3 = ¾ (n+1)

= 3/4(6+1)

= 21/4 = 5.25

Interquartile Range IQR = Q3 - Q1

= 5.25 - 1.75 = 3.5
