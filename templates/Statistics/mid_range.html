What is Meant by Midrange in Statistics and its Formula?

In Statistics, the arithmetic mean of the maximum and minimum data values is called the midrange of the statistical data set. The other name of the midrange is mid-extreme and it is a measure of central tendency.

Mid Range Formula

The formula to find the mid range of the given statistical data is as follows:

M = Xmax + Xmin / 2

Where,

M is midrange
Xmax is the maximum value in a data set
Xmin is the minimum value in a data set

How to Calculate the Midrange Manually with Steps?

To caluclate midrange, follow the below-listed steps carefully & get the value of the mid range of a data set:

  • First, arrange the given data set in ascending order or descending order.
  • Now, find the highest value and lowest value in the data set.
  • Add up the maximum x value and minimum x value.
  • Divide the result by two and calculate the mid-range value of the statistical data.

Example:

Calculate the midrange for the data set values of 2, 6, 7, 11, 32.

Solution:

Given data set values are 2, 6, 7, 11, 32
The given set is arranged in ascending order
So, the lowest value is 2 and the highest value is 32
The formula to find the midrange is
M = maximum + minimum / 2
M = 32+2 / 2
M = 34/2
M = 17

Hence, the mid range of the data set values ie., 2, 6, 7, 11, 32 is 17.
