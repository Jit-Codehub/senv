What is Variance Formula in Probability & Statistics?

In statistics and probability theory, measuring how far a set of numbers are scattered can be known by the variance formula. The variance formula (σ^2) is the squared variation of values (Xi) of a random variable (X) from its mean (μ). To identify, a variance might be of two types and they are as follows:

  • Variance of a population (denoted by σ^2)
  • Variance of a sample (denoted by s^2)

Variance can be of either grouped or ungrouped data. Let's dive into the further article and look at the variance formulas with solved examples.

Variance Formulas for Ungrouped Data

Population variance Sample variance
  • σ2= Σ(xi - μ)2/N

Here,

σ2 = Variance

xi = ith observation of given data

μ = Population mean

N = Total number of observations (Population size)
  • s2 =Σ(xi - x̅)2/n-1

Here,

s2 = Sample variance

xi = ith observation of given data

x̄ = Sample mean

n = Sample size (or Number of data values in sample)

Variance Formulas for Grouped Data

Population variance Sample variance
  • σ2 = ∑ f (m − x̅)2/ n

Here,

σ2 = Variance

xi = Midvalue of ith class

fi = Frequency of ith class

N = Total number of observations (Population size)
  • s2 = ∑ f (m − x̅)2 / n − 1

Here,

s2 = Sample variance

xi = Midvalue of ith class

fi = Frequency of ith class

n = Sample size (or Number of data values in sample)

Solved Variance Formula Examples

Example 1:

Calculate standard deviation and variance of the population data set {4, 8, 10, 48}?

Solution:

Given population data set is {4, 8, 10, 48}

count n = 4

Mean = μ = (∑i = 1nxi) / n = 70/4 = 17.5

Variance = σ² = [(∑i = 1n (xi - μ)²) / n]

= [(4 - 17.5)² + (8 - 17.5 )² + (10 - 17.5)² + (48 - 17.5)²] / 4

= [(-13.5)² + (-9.5)² + (-7.5)² + (30.5)²] / 4

= 1259 / 4

= 314.75

Standard Deviation = σ = √[(∑i = 1n(xi - μ)²) / n] = √variance

= √[314.75]

= 17.74

Example 2:

Find the variance step by step for the following set of data: 4, 6, 11, 25, 38, 50.

Solution:

Step 1: Add up the numbers in your given data set.

4 + 6 + 11 + 23 + 38 + 50 = 132

Step 2: Square your answer:

132 x 132 = 17424

Now, divide by the number of items. ie, 6 items:

17424/6 = 2904

Step 3: Take your set of original numbers from Step 1, and square them individually this time:

4x4 + 6x6 + 11x11 + 23x23 + 38x38 + 50x50

Add the squares together and the result is 4646

Step 4: Subtract the amount in Step 2 from the amount in Step 3.

4646 – 17424 = -12778

Step 5: Subtract 1 from the number of items in your data set, here:

6 – 1 = 5

Step 6: Divide the number in Step 4 by the number in Step 5. This gives you the variance:

-12778/5 = -2555.6

Step 7: Take the square root of your answer from Step 6. This gives you the standard deviation:

√-2555.6 = 50.55i

The answer is 50.55i.

FAQs on Formulas of Variance

1. How do you calculate the variance?

  1. First, memorize the formulas of variance and then start the process of calculating variance.
  2. To find the variance, calculate the mean of the data set. Add all data values and divide by the sample size n.
  3. Next, calculate the squared difference from the mean for each data value.
  4. Subtract the mean from each data value and square the result and discover the sum of the squared differences.
  5. Finally, consider the respective variance formula and substitute the values in it and calculate the variance of the given data set.

2. What is the shortcut formula for Variance?

For the random variable X with a population mean μ and population variance σ2, the shortcut formula of variance is V[X] = E [X^2] - μ^2.

3. What are variance formulas for population and sample data sets?

Variance formula for a population data set is

σ² = [(∑i = 1n (xi - μ)²) / n]

Variance formula for sample data set is

s² = [(∑i = 1n (xi - xmean)²) / (n - 1)]
