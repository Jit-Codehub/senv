What is the Range?

The range of data in statistics is the difference between the maximum and minimum values. It is significantly affected by the large or small values in the data set. The simple formula to compute range is as follows:

Range = Largest value - Smallest value in the data set

Range = Max(x) - Min(x)

Steps to Find Range of Data Set

Following are the steps to calculate the range of data set easily. Go through those guidelines to obtain the answer.

  • Get the set of data.
  • Find the minimum and maximum values in the data set.
  • Subtract minimum value from the maximum value to know the range.

Example:

Question: Calculate the range of the data set {2, 4, 6, 8, 10, 14, 15, 18, 25, 29, 1, 35, 42, 58}?

Solution:

Given data is {2, 4, 6, 8, 10, 14, 15, 18, 25, 29, 1, 35, 42, 58}

Largest value is 58

samllest value is 1

Range = Largest value - Smallest value in the data set

Range = 58 - 1

= 57

Therefore, the data set range is 57.

Frequently Asked Question's on Range Calculator

1. How do you calculate the range?

To calculate the value of range within a data set, subtract the lowest number from the highest number.

2. Why do we calculate range?

The range is calculated to estimate how different and varied the numbers are in the data set.

3. What is range explain?

In statistics, the range is defined as the difference between the maximum and minimum values within a set of numbers.

4. Find the range of the data set 15, 2, 3, 4, 13, 17, 14, 22?

Given set of data is 15, 2, 3, 4, 13, 17, 14, 22

The smallest value is 2 and the largest value is 22

The range is the difference between the largest and smallest value.

So, range = 22 - 2 = 20.
