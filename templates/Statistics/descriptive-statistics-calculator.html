What is meant by Descriptive Statistics?

Descriptive Statistics condense certain features using numerical calculations. Two basic types of descriptive statistics will be measures of central tendency and the measures of dispersion. Some of the examples of descriptive statistics will be the mean, median, mode, variance, maximum, minimum, midrange, Standard deviation, relative standard deviation, root mean square, sum, size count, sum of squares.

Descriptive Statistics Formulas

We will see the formulas for each and every representation for a sample or population.

Let's say the data set is x1, x2, x3, x4, . . . xn.

1. Minimum

Arrange the data set from low to high i.e x1 ≤ x2 ≤ x3 ≤ x4 ≤ . . . xn

Minimum value = xmin = min(xi)i=1n

2. Maximum

Arrange the data set from low to high i.e x1 ≤ x2 ≤ x3 ≤ x4 ≤ . . . xn

Maximum value = xmax = max(xi)i=1n

3. Range

Range is described as the difference between maximum and minimum.

Range = xmax - xmin

4. Sum

Sum is the total of all the data items in the data set.

Sum = ∑i = 1nxi

5. Size, Count

Size or count is the number of values in the data set.

Size = n = count(xi)i = 1n

6. Mean

Mean is defined as sum of all the data divided by the size of the data in the data set.

For population

μ = (∑i = 1nxi) / n = Sum / size

For sample

xmean = (∑i = 1nxi) / n

7. Median

Arrange the given data from low to high. Median is the value that separates the first half of the ordered sample data from the other half. If n is odd, then mid value is the median. If n is even then the median is the average of 2 values at the center.

If n is odd then the median value at position p is

p = (n + 1) / 2

xmedian = xp

If n is even, then the median becomes the average of two mid values at p and p+1 positions

p = n/2

xmedian = ( xp + xp+1) / 2

8. Mode

Mode is the values that occur most often in the data set. A data set can have more than one mode or don't have any mode.

9. Standard Deviation

For population

σ = √[(∑i = 1n (xi - μ)²) / n]

For sample

s = √[(∑i = 1n (xi - xmean)²) / (n - 1)]

10. Variance

For population

σ² = [(∑i = 1n (xi - μ)²) / n]

For sample

s² = [(∑i = 1n (xi - xmean)²) / (n - 1)]

11. Relative Standard Deviation

For population

RSD = [(100 * σ) / μ]%

For sample

RSD = [(100 * s) / xmean]%

12. Midrange

Mid range is the average of minimum and maximum values.

MR = (xmin + xmax) / 2

13. Quartiles

Quartiles separates given data set into four sections. Second quartile Q2 is the median, that divides ordered data set into two half values. first quartile median is Q1 of the lower half that not includes Q2. Third quartile Q3 is the median of the higher half that does not include Q2.

14. Interquartile Range

Range from Q1 to Q3 is interquartile range (IQR)

IQR = Q3 - Q1

15. Outliers

Fence for upper = Q3 + 1.5 x IQR

Fence for lower = Q1 - 1.5 x IQR

16. Sum of Squares

For Population

SS = ∑i = 1n [xi - μ]²

For Sample

SS = ∑i = 1n (xi -xmean)²

17. Mean Absolute Deviation

For Population

MAD = (∑i = 1n |xi - μ|) / n

For Sample

MAD = (∑i = 1n |xi -xmean|) / n

18. Root Mean Square

RMS = √[(∑i = 1n (xi)²) / n ]

19. Standard Error of the Mean

For Population

SEμ = (σ / (√n))

For Sample

SExmean = (s / (√n))

20. Skewness

For Population

γ1 = (∑i = 1n (xi - μ)3) / (n * σ3)

For Sample

γ1 = [(n) / ((n - 1) (n - 2)] [∑i = 1n ((xi - xmean) / s)3]

21. Kurtosis

For Population

β2 = (∑i = 1n (xi - μ)4) / (n * σ4)

For Sample

β2 = [[n (n +1)] / ((n - 1) (n - 2) (n - 3))] [∑i = 1n ([xi - xmean]4 / s)]

22. Kurtosis Excess

For Population

α4 = [((∑i = 1n (xi - μ)4) / (n * σ4)) - 3]

For Sample

α4 = {[(n (n + 1)) / ((n - 1)(n - 2)(n - 3)] [∑i = 1n((xi - xmean) / s)4] - [(3(n-1)²) / ((n - 2) (n - 3))]

23. Coefficient of Variation

For Population

CV = σ / μ

For Sample

CV = s / xmean
