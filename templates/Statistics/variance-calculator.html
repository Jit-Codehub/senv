What is Meant by Variance?

In statistics, variance is a measure of the variability of the values in the dataset. Mathematically, it is defined as the average squared difference from the mean for all values. The square root of variance is called the standard deviation.

Where,

x̅ is the mean of all data points

xi is the value of one data point

s² is the sample variance

n is the number of observations

Other important variance formulas are listed here.

  1. Size of the data set Size = n = count(xi)i = 1n
  2. Mean for sample data set xmean = (∑i = 1nxi) / n
  3. Mean for population data set μ = (∑i = 1nxi) / n = Sum / size
  4. Sum of squares for sample data set SS = ∑i = 1n (xi -xmean)²
  5. Sum of squares for population data set SS = ∑i = 1n (xi - μ)²
  6. Variance for population data set σ² = [(∑i = 1n (xi - μ)²) / n]
  7. Standard deviation for population data set σ = √[(∑i = 1n (xi - μ)²) / n]
  8. Standard deviation for sample data set s = √[(∑i = 1n (xi - xmean)²) / (n - 1)]

How to calculate Variance?

Check out the below listed easy process on finding the sample variance of data set. The guidelines are along the lines

  • At first, get the data set.
  • Calculate the mean of data set by dividing the sum of all observations by the number of observations.
  • Subtract mean from each data point in the set.
  • Add the obtained values.
  • Divide the sum by the number of observations minus 1 to check the sample variance.
  • Otherwise, Substitute the values in the formula to get the answer.

Example

Question: Calculate variance, sum of squares, standard deviation, mean of the sample data set {1, 6, 8, 5}?

Solution:

Given sample data set {1, 6, 8, 5}

count n = 4

Mean = xmean = (∑i = 1nxi) / n

= (1+6+8+5) / 4

= 20 / 4 = 5

Sum of Squares = ss = ∑i = 1n (xi -xmean)²

= (1 - 5)² + (6 - 5)² + (8 - 5)² + (5 - 5)²

= (-4)² + (1)² + (3)² + (0)²

= 16 + 1 + 9 + 0 = 26

Variance = s² = [(∑i = 1n (xi - xmean)²) / n]

= [((1 - 5)² + (6 - 5)² + (8 - 5)² + (5 - 5)²] / 4

= [(-4)² + (1)² + (3)² + (0)²] / 4

= (16 + 1 + 9 + 0) / 4 = 26 / 4

= 6.5

Standard Deviation = s = √[(∑i = 1n (xi - xmean)²) / n]

= √[((1 - 5)² + (6 - 5)² + (8 - 5)² + (5 - 5)²] / 4]

= √[[(-4)² + (1)² + (3)² + (0)²] / 4]

= √[(16 + 1 + 9 + 0) / 4]

= √[26 / 4]

= √[6.5]

= 2.54

FAQ's on Variance Calculator

1. What is the purpose of calculating variance?

The main purpose of calculating sample variance is to check how individual members relate to each other in a data set, rather than using techniques like arranging numbers into quartiles.

2. How do you calculate variance quickly?

To find the value of variance quickly, enter the data set values in the mentioned input fields of the calculator and press the calculate button.

3. Is variance a standard deviation?

The variance is the average of the squared differences from the mean. Whereas standard deviation is the square root of the variance.

4. What is the formula of variance?

The variance formula is σ² = [(∑i=1n (xi - μ)²) / n]. Where n is the number of items in data set, μ is the mean.
