What is Percentile?

Percentile is a value in the data set where it one particular value is compared with other values in the data set. This percentile can be splitted into two parts. Lower part contains the set of data 'p' and the upper part contains remaining data '100-p'.

Formula for percentile

i = (p/100) * n

Where,

p is the percentile.

n is the number of values in the given data set.

i is the integer or non-integer.

How do you Calculate Percentile?

As we want to calculate the kth percentile manually we need to follow some basic steps to do it easily without much effort.

  • Make a note of the given data in the question.
  • Arrange all the data from lowest to highest(Ascending order).
  • Now, apply the formula and substitute the values in it.
  • If i is not an integer, then you need to round up. otherwise you need to find the average of the positions i and i+1.

How to Find the Percentile?

Question 1: Find the 20th percentile for the given set of values 8, 2, 5, 6, 9?

Solution

Given set of values is 8, 2, 5, 6, 9

Arrange the values in ascending order

2, 5, 6, 8, 9

p= 20, n= 5, i=?

Formula: i = (p/100) * n

i = (20/100) * 5

= 1

As, i is an integer, the average of 0th and 1st positions in the ordered data set is 8 and 2.

Thus, the 20th percentile of the given set is (2+5)/2 =7/2 = 3.5.
