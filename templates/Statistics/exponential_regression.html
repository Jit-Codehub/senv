What is Exponential Regression Model?

The exponential regression aims to find an exponential curve that suits the given set of data points. It is similar to linear regression, where a straight line best fits the quadratic equation. There are a number of situations where the data follows neither parabola nor the straight line, but an exponential curve seems to be right. Here, the curve grows slowly in the starting and then accelerate rapidly. Some of the examples are radioactive decay, investment growth.

Suppose take x as the explanatory variable, y as the response variable.

The exponential regression equation will be y = a * bx.

Where,

a ≠ 0

b > 0 and b ≠ 1

The coefficients are selected for the equation that corresponds to the exponential curve of best suits the data set (x1, y1), . . (xn, yn)

Steps to Find Exponential Regression

The following are the simple steps to calculate the exponential regression equation that suits the given data set.

  • First of all, transform the data so that it allows a linear model.
  • Solve that linear model using the method of least squares.
  • Again transform the data back to the original form.
  • Substitute the data point values in formula y = exp(c) * exp(m * x).

Exponential Regression Equation Formulas

We know that exponential regression model equation should be in the form of y = a * bx. And we have to determine how it is derived.

Take a logarithm for both sides of equation

log(y) = log(a * bx)

Using the logarith property log(y) = log(a) + log(bx)

log(y) = log(a) + x * log(b)

Here, log(y) is in the form of linear function of x with log(b) as slope and log(A) as intercept.

Find the best-fit line for (x, log(y))

log(y) = c + x * m

So, log(A) = c and log(b) = m

Therefore, a = exp(c) and b = exp(m)

So, exponential regression equation is y = exp(c) * exp(m * x).

FAQs on Exponential Regression Model Calculator

1. How do you calculate exponential regression by hand?

To calculate the linear regression equation, we need to transform the data to allow a linear model. Find the linear model and transform data back along with the model to the original form.

2. Is exponential regression linear?

Exponential regression finds the equation of the exponential function(y=abx) that best fits the data. In linear regression, we find the equation (y=b+mx) that fits the best data. Therefore, exponential regression is non-linear.

3. What is the exponential regression model?

The exponential regression equation is y = a * bx. Where x is the explanatory variable, y is the response variable, and coefficients a ≠ 0, b > 0, b ≠ 1.

4. What is exponential regression used for?

Exponential regression is used to model situations where growth starts slowly and then accelerates rapidly without bound or decay begin rapidly and slows down to get closer to zero.
