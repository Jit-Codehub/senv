What is Standard Deviation in Statistics?

In statistics, the measure of the amount of variation or dispersion of a set of values is known as a standard deviation. Also, the standard deviation is the square root of the variance. You can see two types of standard deviations ie., low standard deviation and high standard deviation. The sign for Standard Deviation is 'σ' (the Greek letter sigma).

List of Standard Deviation Formulas

The table presented below will make you learn all the formulas related to the concept of standard deviation. By memorizing these formulas, you can easily calculate standard deviation problems.

S.No. Formulas
1. Sum

Sum = ∑i = 1nxi
2. Size Count = n = count(xi)i = 1n
3. Mean

For population data set

μ = (∑i = 1nxi) / n = Sum / size

For sample data set

xmean = (∑i = 1nxi) / n
4.

Sum of Squares

For a Population

SS = ∑i = 1n (xi - μ)²

For a Sample

SS = ∑i = 1n (xi -xmean)²
5. Standard Deviation

For a population

σ = √[(∑i = 1n (xi - μ)²) / n]

For a sample

s = √[(∑i = 1n (xi - xmean)²) / (n - 1)]
6. Variance

Variance formula for a a population is

σ² = [(∑i = 1n (xi - μ)²) / n]

For a sample is

s² = [(∑i = 1n (xi - xmean)²) / (n - 1)]

Where
xi is an individual value
μ is the mean/expected value
n is the total number of values

How to Calculate Standard Deviation Easily?

Here are the simple steps that you should follow to solve the standard deviation manually. These steps are so convenient and comfortable while standard deviation calculation. So, make use of them without any fail:

  • First, you should consider your population or sample data set.
  • Next, calculate the mean of the data set by applying the formula.
  • Now, find the difference between each data value in the set and the mean.
  • At this step, the result will be squared and added all the values.
  • Finally, divide the acquired answer by the total number of values to obtain the standard deviation.

Example:

Find standard deviation of the population data set: {5, 15, 25, 35, 45}

Solution:

Given data set is {5, 15, 25, 35, 45}

Firstly, we have to find the arithmetic mean of the given data set

Here, Count = 5, Sum = 5+15+25+35+45 = 125

Hence, Mean = Sum / Count

= 125/5

= 25

Sum of Squares = SS = ∑i = 1n (xi - μ)²

= (5-25)²+(15-25)²+(25-25)²+(35-25)²+(45-25)²

= -20²+-10²+0²+10²+20²

= 400+100+0+100+400

= 1000

Variance = sum of difference²(squares) / count

= 1000/5 = 200

Standard Deviation = √Variance

σ = √[(∑i = 1n (xi - μ)²) / n]

= √200

= 14.1421356237
