What is meant by 5 Number Summary?

In general, Five Number Summary is nothing but the visual representation using the boxplot for given set of data. For given values five number summary denotes the maximum value, minimum value, fist quartile, second quartile, third quartile, etc. All these values can be found by arrangig the given data set in ascending order.

  • Minimum (Min) - the smallest observation
  • Maximum (Max) - the largest observation
  • Median (M) - the middle term
  • First Quartile (Q1) - the middle term of values below the median
  • Third Quartile (Q3) - the middle term of values above the median

How do you calculate the Five Number Summary?

In order to compute the five number summary of a data set: i.e. minimum value, maximum value, first, second and third quartiles you need to follow certain steps and they are along the lines

  • Firstly, Write down the given list of numbers.
  • Arrange the given numbers in ascending order and then find the minimum and maximum values from the set.
  • Next step is to find the median i.e. middle term from the list. If the set of numbers is odd then median is middle number. In case of even data set determine the sum of two middle values and then divide by 2 to obtain the median.
  • Place the values in the paranthesis other than the median.
  • Calculate the median in the lower half to get the 1st quartile and likewise find out the median in the upper half and it is the third quartile or upper quartile.
  • Make a note of all the values you have found to make a five number summary.

Five Number Summary Example Calculations

Example 1: Calculate the five number summary of unarranged data set listed below: 4, 13, 19, 2, 62, 3, 8, 11, 9

Solution:

  • Firstly, write down the given set of numbers i.e. 4, 13, 19, 2, 62, 3, 8, 11, 9
  • Sort the given set of numbers and arrange them in ascending order we have 2, 3, 4, 8, 9, 11, 13, 19, 62
  • Now, in the ascending order arrangement find out the minimum value and maximum value. Min = 2, Max = 62
  • Let us find the median. As the data set is odd the middle term is the median i.e. 9
  • Leaving aside the median place all other values in parathensis i.e. (2, 3, 4, 8) 9 (11, 13, 19, 62)
  • Find the median in the lower half and upper half to get the 1st, 2nd quartile values. Q1 =(3+4)/2 = 3.5, Q3=(13+19)/2 = 16
  • Now that you have found all the values of the 5 number set write them i.e. 1st Quartile = 3.5, Median or Second Quartile = 9, Third quartile = 16, Minimum = 2, Maximum = 62

FAQs on Five Number Summary Calculator

1. What is the five summary in math?

A five-number summary simply involves the minimum value, maximum value, first quartile, second quartile, third quartile.

2. How do you find Q1 and Q3 in Five Number Summary?

To find Q1 and Q3 in Five Number Summary simply arrange the given set of numbers in ascending order. Then, find the median of lower half to get Q1 or First Quartile and median of upper half to obtain the Upper or Third quartile Q3.

3. What is Q1 in Five Number Summary?

Q1 in Five Number Summary is nothing but the First Quartile.

4. Is Median a part of Five Number Summary>

Yes, Median is a part of Five Number Summary.
